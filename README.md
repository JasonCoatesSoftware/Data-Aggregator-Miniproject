# Car Sales Data Aggregator
This is a Java backend application that reads car sales data from different file formats (currently .csv and .json), standardizes it, and calculates the total value of cars sold. In the future there will also be more functions, but this code is the strong foundation. This project uses a thread pool to process multiple files simultaneously, and a generic parser interface that can be used to handle other data types in the future. To utilize, please place your files in the /data directory and run Main.java.
